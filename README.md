# vue-package-template

vue-package-template 基于`vue3`实现的`vue组件`、`vue插件`打包项目模板，由`rollup`作为打包。

## 安装

1. 启动`example`

```
$ yarn dev
```

2. 编译`src`中的代码

```
$ yarn build
```

3. 打包`src`中的代码

```
$ yarn start
```

4. 上传至`npm`

```
$ yarn pub
```
